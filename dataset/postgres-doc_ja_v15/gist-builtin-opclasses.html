歴史的な理由から、inet_ops演算子クラスは型inetとcidrのデフォルトクラスではありません。 これを使用するには、CREATE INDEXでクラス名を指定します。 例えば、以下のようにします。 

CREATE INDEX ON my_table USING GIST (my_inet_column inet_ops);
