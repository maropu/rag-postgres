SPI_cursor_find

SPI_cursor_find — 既存のカーソルを名前で検索する

概要

Portal SPI_cursor_find(const char * name)

説明

SPI_cursor_findは既存のカーソルを名前で検索します。 これは主に、他の何らかの関数でテキストとして返されたカーソル名の名前解決の際に使用されます。

引数

const char * name

ポータルの名前

戻り値

指定された名前のポータルへのポインタ。見つからない場合はNULLです。
